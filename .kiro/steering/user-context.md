# ユーザーコンテキスト

## プロジェクト詳細
- **プロダクト**: AWS プラクティショナー問題集アプリ
- **技術スタック**: PWA (Progressive Web App)
- **対象デバイス**: iPhone Safari
- **データベース**: IndexedDB (ローカル)
- **問題形式**: マークダウン

## ユーザー好み
- 日本語でのコミュニケーション
- PWA技術の使用
- シンプルで保守性の高いコード
- 段階的な開発アプローチ

## アプリケーション要件
- 一問一答形式
- 選択肢選択 → 決定ボタン → 結果表示 → 解説表示 → 次の問題
- **ラウンド制の再出題機能**：すべての問題を一通り終えてから、間違えた問題のみで新しいラウンドを開始
- 進捗表示（現在のラウンドの問題数、正解数）
- マークダウン形式での問題追加機能
- 画像・リンク対応

## 開発哲学
「小さく始めて、賢く構築する」- 基本機能から始めて段階的に拡張
