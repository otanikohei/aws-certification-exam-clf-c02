# 災害復旧計画

## 問題
クラウド実践者は災害復旧計画を策定しており、複数の地理的エリア間でデータを複製する予定です。  
これらの要件を満たすには、AWS クラウドのどのコンポーネントを使用する必要がありますか?

## 選択肢
1. AWS アカウント
2. AWS リージョン
3. アベイラビリティ・ゾーン
4. エッジ・ロケーション

## 正解
2

## 解説
AWS でいうところの「災害」は、データセンターの障害ではなく、東京が沈没するぐらいの大災害を意味します。  
なので、災害復旧計画には、リージョン対策が必要です。

データセンター -> アベイラビリティ・ゾーンで十分  
地理的エリア -> リージョン対策が必要

リージョンの中にアベイラビリティ・ゾーンがあります。
エッジロケーションは、アクセスする人に近い場所にあるデータセンターのことで、アベイラビリティ・ゾーンよりもたくさんあります。

2025 年 12 月時点  
- リージョン数: 38
- アベイラビリティゾーン数: 117  
- エッジロケーション: 700 以上
