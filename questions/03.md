# サービスの制限

## 問題
ユーザーが VPC サービスの制限の増加をリクエストする必要があるかどうかを判断するのに役立つのは次のどれですか。

## 選択肢
1. AWS Health Dashboard
2. AWS Trusted Advisor
3. AWS Cost and Usage Report
4. AWS Service Catalog

## 正解
2

## 解説
AWS Trusted Advisor には、サービスの制限（クオータ）をチェックする機能があります。

![AWS Trusted Advisor画面](images/03-01.png)

**サービスの制限（クオータ）とは**  
AWS はボタンひとつでリソースが作れてしまうので、操作を間違えると高額課金されてしまいます。  
意図しないリソースの作成を制限するために、ウオータという利用制限機能があります。
