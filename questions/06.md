# Amazon Macie - PII検出

## 問題
個人を特定できる情報 (PII) が Amazon S3 バケット内に保存されている場合に検出結果を生成できる AWS のサービスはどれですか?

## 選択肢
1. Amazon GuardDuty
2. Amazon Macie
3. Amazon Inspector
4. AWS Trusted Advisor

## 正解
2

## 解説
**Amazon Macie** は、機械学習を使用してS3内の機密データを自動的に検出・分類するデータセキュリティサービスです。

### Amazon Macieの主な機能

#### 1. PII（個人識別情報）の検出
- **クレジットカード番号**
- **社会保障番号**
- **パスポート番号**
- **運転免許証番号**
- **メールアドレス**
- **電話番号**

#### 2. 機密データの分類
- 機械学習とパターンマッチングを使用
- カスタムデータ識別子の作成も可能
- データの機密度レベルを自動判定

#### 3. セキュリティ検出結果の生成
- 機密データが発見された場合にアラートを生成
- AWS Security Hubとの統合
- CloudWatchイベントとの連携

### 他の選択肢との違い

#### A. Amazon GuardDuty
- **脅威検出サービス**
- 悪意のあるアクティビティや異常な動作を検出
- PII検出は対象外

#### C. Amazon Inspector
- **アプリケーションセキュリティ評価サービス**
- EC2インスタンスやコンテナの脆弱性スキャン
- S3内のデータ内容は検査しない

#### D. AWS Trusted Advisor
- **ベストプラクティス推奨サービス**
- コスト最適化、パフォーマンス、セキュリティの推奨事項
- データ内容の検査は行わない

### 実用例
```
S3バケット内のファイルに以下が含まれている場合：
- 顧客データベースのCSVファイル
- クレジットカード情報を含むログファイル
- 個人情報を含むドキュメント

→ Macieが自動的に検出してアラートを生成
```
