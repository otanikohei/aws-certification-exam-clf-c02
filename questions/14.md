# 問題 14

新しいマルチアカウントAWS環境をセットアップし、大規模に管理するためのシンプルな方法を提供するAWSサービスはどれですか？

- A) AWS Trusted Advisor
- B) AWS Security Hub
- C) AWS Control Tower
- D) AWS Resource Access Manager

## 正解

**C**

## 解説

### A) AWS Trusted Advisor ❌

**不正解**。Trusted Advisorは、AWS環境を検査し、改善すべき領域に関する推奨事項を提供するために使用されます。Trusted Advisorは、マルチアカウント環境のセットアップと管理には使用されません。

> 📚 [Trusted Advisorについて詳しく学ぶ](https://aws.amazon.com/premiumsupport/technology/trusted-advisor/)

---

### B) AWS Security Hub ❌

**不正解**。Security Hubは、AWS環境がセキュリティ標準のベストプラクティスとどのように比較されるかの包括的なビューを提供します。Security Hubは、優先度の高いセキュリティアラートや検出結果も提供します。Security Hubは、マルチアカウント環境のセットアップと管理には使用されません。

> 📚 [Security Hubについて詳しく学ぶ](https://aws.amazon.com/security-hub/)

---

### C) AWS Control Tower ✅

**正解**。AWS Control Towerは、複数のAWSアカウントをセットアップ、管理、ガバナンスする機能を提供します。

> 📚 [AWS Control Towerについて詳しく学ぶ](https://aws.amazon.com/controltower/)

---

### D) AWS Resource Access Manager ❌

**不正解**。Resource Access Manager（AWS RAM）は、AWS Organization内の任意のAWSアカウントとAWSリソースを共有する機能を提供します。

> 📚 [AWS RAMについて詳しく学ぶ](https://aws.amazon.com/ram/)
