# 問題 20

企業は60TBのデータを安全な方法でAWSクラウドに転送する必要があります。この要件を満たすために、企業は何を使用すべきですか？

- A) AWS Snowball Edge
- B) Amazon Elastic Block Store (Amazon EBS)
- C) Amazon Elastic File System (Amazon EFS)
- D) Amazon S3

## 正解

**A**

## 解説

### A) AWS Snowball Edge ✅

**正解**。Snowball Edgeは、オンボードストレージとコンピューティング能力を備えたハードウェアデバイスです。Snowball Edgeは、ローカル環境とAWSクラウド間でデータを転送します。Snowball Edgeは、ローカル処理とエッジコンピューティングワークロードも実行できます。Snowball Edgeは、テラバイト規模のデータ転送をサポートします。

> 📚 [Snowball Edgeについて詳しく学ぶ](https://aws.amazon.com/snowball/)

---

### B) Amazon Elastic Block Store (Amazon EBS) ❌

**不正解**。Amazon EBSはデータを転送しません。Amazon EBSは、Amazon EC2インスタンスで使用できるブロックレベルのストレージボリュームを提供します。

> 📚 [Amazon EBSについて詳しく学ぶ](https://aws.amazon.com/ebs/)

---

### C) Amazon Elastic File System (Amazon EFS) ❌

**不正解**。Amazon EFSはデータを転送しません。Amazon EFSは、AWSクラウドサービスとオンプレミスリソースで使用できるサーバーレスの弾力的なファイルシステムを提供します。

> 📚 [Amazon EFSについて詳しく学ぶ](https://aws.amazon.com/efs/)

---

### D) Amazon S3 ❌

**不正解**。Amazon S3はデータを転送しません。Amazon S3は、あらゆる量のデータに対してスケーラビリティ、データ可用性、セキュリティ、パフォーマンスを提供するオブジェクトストレージサービスです。

> 📚 [Amazon S3について詳しく学ぶ](https://aws.amazon.com/s3/)
