# 問題 28

静的画像を保存するための最もコスト効率の良いAWSサービスはどれですか？

- A) Amazon S3
- B) Amazon DynamoDB
- C) AWS Snowball Edge
- D) Amazon Elastic Block Store（Amazon EBS）

## 正解

**A**

## 解説

### A) Amazon S3 ✅

**正解**。Amazon S3は、使用した分だけ支払うため、静的画像を保存するための最もコスト効率の良いオプションです。

> 📚 [Amazon S3の料金について詳しく学ぶ](https://aws.amazon.com/s3/pricing/)

---

### B) Amazon DynamoDB ❌

**不正解**。DynamoDBはNoSQLキーバリューデータベースです。DynamoDBを使用してJSONドキュメントを保存できますが、DynamoDBに画像を保存することはできません。

> 📚 [DynamoDBについて詳しく学ぶ](https://aws.amazon.com/dynamodb/)

---

### C) AWS Snowball Edge ❌

**不正解**。Snowball Edgeは、ある物理的な場所からAWSにデータを転送するために使用できる物理デバイスです。

> 📚 [Snowball Edgeについて詳しく学ぶ](https://aws.amazon.com/snowball/)

---

### D) Amazon Elastic Block Store（Amazon EBS） ❌

**不正解**。Amazon EBSの料金は、プロビジョニングされたストレージの量と必要なパフォーマンスレベルに基づいています。Amazon EBSを使用して静的画像を保存できますが、GB単価ベースでは、画像の保存にはAmazon S3の方がコスト効率が良いです。

> 📚 [EBSの料金について詳しく学ぶ](https://aws.amazon.com/ebs/pricing/)
