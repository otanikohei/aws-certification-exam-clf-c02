# 問題 39

企業はAWSリソースを監査する必要があります。企業はリソースに加えられた変更を文書化する必要があります。これらの要件を満たすAWSサービスはどれですか？

- A) AWS Artifact
- B) AWS Config
- C) Amazon Inspector
- D) Amazon CloudWatch

## 正解

**B**

## 解説

### A) AWS Artifact ❌

**不正解**。AWS Artifactは、AWSのセキュリティおよびコンプライアンスレポートと契約にアクセスするための中央の場所を提供します。AWS Artifactは、AWSリソースへの変更を監査または文書化しません。

> 📚 [AWS Artifactについて詳しく学ぶ](https://aws.amazon.com/artifact/)

---

### B) AWS Config ✅

**正解**。AWS Configは、AWSリソースのインベントリと設定履歴の監査レポートを提供できるフルマネージドサービスです。

> 📚 [AWS Configについて詳しく学ぶ](https://aws.amazon.com/config/)

---

### C) Amazon Inspector ❌

**不正解**。Amazon Inspectorは、Amazon EC2インスタンス、Amazon Elastic Container Registry（Amazon ECR）コンテナイメージ、AWS Lambda関数を含むAWSワークロードをスキャンします。Amazon Inspectorは、これらのワークロードをソフトウェアの脆弱性と意図しないネットワーク露出についてスキャンします。Amazon Inspectorは、AWSリソースへの変更を監査または文書化しません。

> 📚 [Amazon Inspectorについて詳しく学ぶ](https://aws.amazon.com/inspector/)

---

### D) Amazon CloudWatch ❌

**不正解**。CloudWatchは、AWSリソースをリアルタイムで監視するのに役立つメトリクスを収集および追跡します。CloudWatchは、AWSリソースへの変更を監査または文書化しません。

> 📚 [CloudWatchについて詳しく学ぶ](https://aws.amazon.com/cloudwatch/)
