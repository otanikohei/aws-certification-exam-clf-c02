# EC2インスタンスタイプ

## 問題
Amazon EC2のインスタンスタイプについて正しい説明はどれですか？

## 選択肢
1. すべてのインスタンスタイプは同じ性能を提供する
2. インスタンスタイプによってCPU、メモリ、ストレージ、ネットワーク性能が異なる
3. インスタンスタイプは起動後に変更できない
4. 無料利用枠ではすべてのインスタンスタイプが利用可能

## 正解
2

## 解説
Amazon EC2では、様々なワークロードに最適化された複数のインスタンスタイプが提供されています。

主なインスタンスファミリー：
- **汎用（T3、M5など）**: バランスの取れた性能
- **コンピューティング最適化（C5など）**: CPU集約的なワークロード
- **メモリ最適化（R5など）**: メモリ集約的なワークロード
- **ストレージ最適化（I3など）**: 高いディスクI/O性能

インスタンスタイプは停止後に変更可能です。
