# AWS IAMの基本概念

## 問題
クラウド実践者は災害復旧計画を策定しており、複数の地理的エリア間でデータを複製する予定です。  
これらの要件を満たすには、AWS クラウドのどのコンポーネントを使用する必要がありますか?

## 選択肢
1. AWS accounts
2. AWS Regions
3. Availability Zones
4. Edge locations

## 正解
2

## 解説
AWS でいう災害復旧は、例えば日本が半分沈むぐらいの大災害を指します。  
なので、災害対策として、**大阪リージョン** とか、**シンガポールリージョン**みたいな、かなり離れた場所にデータを複製する必要があります。
